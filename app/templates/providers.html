How It Works

API keys are already set. The app reads them from Railway environment variables:

  • OPENROUTER_API_KEY — for OpenRouter models
  • OPENAI_API_KEY — for OpenAI models
  • ANTHROPIC_API_KEY — for Anthropic models

Just pick a model and assign it to a role.

  • ✍️ Writer (Chat) — generates the actual responses the auto-pilot sends
  • 🛡️ Supervisor — reviews messages before sending, analyzes bans
  • 🔄 Fallback — used when the main provider fails

💡 Use a fast/cheap model for Supervisor (e.g., gpt-4o-mini) and a better one for Writer (claude-sonnet-4, gpt-4o, deepseek-chat).
